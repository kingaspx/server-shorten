# Shortner URL

This project is a URL shortener. The client version will be available below.

## Run Locally

Clone the project

```bash
  git clone https://github.com/rodriguesabner/server-shorten
```

Go to the project directory

```bash
  cd server-shorten
```

Install dependencies

```bash
  npm i
  //or
  yarn
```

Start the server

```bash
  npm run start
```
